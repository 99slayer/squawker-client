#### Features
- Account creation
- Account updating
- Homepage timeline
- User profiles
- User profile pictures and profile headers
- User followers and following
- Post/comment creation (including images!)
- Post/comment editing
- Post/comment quoting and reposting
- Post/comment liking
- Guest login

#### Client Tech Stack
- HTML5
- CSS
- Tailwind
- React
- TypeScript
- Supabase
